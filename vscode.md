# about vscode

## Extensions

### Render Line Endings

Render CR/LF/CRLF respectively.  
Can be configured as below:  

    {
        "editor.renderWhitespace": "all",
        "code-eol.newlineCharacter": "¬",
        "code-eol.returnCharacter" : "¤",
        "code-eol.crlfCharacter"   : "¤¬",
    }
